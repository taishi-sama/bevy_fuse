FUSE-based filesystem where files stored directly in entities, with metadatas and file contents stored in corresponding components.
Instantly displays these entities on screen using Bevy's Sprite component, pushing them separately using Avian physics engine.

### 
Post with video demonstration: https://www.reddit.com/r/rustjerk/comments/1q0093a/tired_of_your_absolutely_boring_inmemory/

### How To Run
First build it
```
cargo build --release
```
Then run it
```
./target/release/bevy_fuse --mount-point <path_to_desired_mountpoint>
```
